2.1 Практические задачи

Задача 1: Создание тестов с ChatGPT

Цель: Научиться использовать ИИ для автоматической генерации вопросов по учебному материалу

  1. Сформулируйте тему задания (например, "Фотосинтез")
  2. Введите запрос в ChatGPT: "Сгенерируй 5 тестовых вопросов по теме [тема] с вариантами ответов"
  3. Проанализируйте полученные вопросы, отредактируйте при необходимости
  4. Используйте в учебном процессе

Полезные инструменты:

ChatGPT, Quizizz, Quizlet

Задача 2: Автоматизированная проверка сочинений

Цель: Ознакомить учащихся с инструментами проверки грамматики и стиля

  1. Ученики пишут эссе на заданную тему
  2. Загружают текст в Grammarly или Hemingway Editor
  3. Анализируют предложенные исправления
  4. Улучшают текст на основе рекомендаций

2.2 Упражнения для развития цифровой грамотности

Упражнение 1: Проверка достоверности информации

  1. Дайте ученикам спорное утверждение (например, "ИИ может полностью заменить учителей")
  2. Попросите проверить его с помощью Perplexity AI или Elicit
  3. Обсудите найденные источники и их надежность

Упражнение 2: Создание учебного контента

Пример: Создание карточек по биологии с помощью ИИ

  1. Выберите тему ("Клеточное строение организмов")
  2. Используйте ChatGPT: "Создай 10 учебных карточек по теме [тема]"
  3. Оформите в Quizlet или Anki
  4. Поделитесь с классом

2.3 Применение ИИ в исследовательской деятельности

Проект: Анализ данных с Wolfram Alpha

Цель: Научиться использовать ИИ для обработки статистических данных

  1. Соберите данные (например, результаты опроса)
  2. Введите в Wolfram Alpha для анализа
  3. Постройте графики и диаграммы
  4. Сформулируйте выводы

Альтернативные инструменты:

Google Sheets с AI-дополнениями, Microsoft Excel с Ideas
